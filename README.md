# Telecom Threat Hunter

## Overview
AI-assisted defensive threat hunting platform for telecom and cloud environments using Python, Ollama, and Streamlit.

## Features
- Cloud identity analysis
- Suspicious login detection
- OAuth abuse analysis
- MITRE ATT&CK mapping
- AI-generated analyst summaries
- Local/private LLM inference

## Stack
- Python
- Ollama
- Streamlit
- Llama3
- Prompt Engineering

## Purpose
This project demonstrates how local LLMs can assist CTI and SOC analysts by accelerating triage and providing contextual threat analysis for suspicious cloud activity.

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
